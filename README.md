## 📋 版本规则

版本号格式：`vX.Y.Z`

| 变更 | 规则 |
|------|------|
| Z +1 | Bug修复（集中1-2天测试通过后） |
| Y +1 | 新功能上线 |
| X +1 | 重置/不兼容性变更 |

每次发布前需与老危确认版本号，并同步更新开发计划版本记录。
